# Airline-Passenger-Satisfaction-Analysis
Interactive Tableau dashboard analyzing airline passenger satisfaction, service quality, customer loyalty and flight delay impacts using data-driven insights.

## Overview
This project analyzes airline passenger satisfaction data to identify key factors influencing customer experience, including flight delays, travel class, customer loyalty, and service quality.

## Tools Used
- Tableau
- SQL
- Excel

## Dataset
The dataset contains passenger demographics, travel information, service ratings, flight delays, and satisfaction status.

## Objectives
- Analyze customer satisfaction trends
- Identify factors affecting passenger experience
- Evaluate the impact of delays on satisfaction
- Compare satisfaction across travel classes and customer types

## Key KPIs
- Satisfaction Rate
- Average Departure Delay
- Average Arrival Delay
- Customer Type Distribution

## Key Insights
- Business Class passengers reported higher satisfaction.
- Loyal customers showed greater satisfaction levels.
- Longer delays negatively impacted customer experience.
- Seat comfort and inflight entertainment were major drivers of satisfaction.

## Recommendations
- Reduce flight delays.
- Improve Economy Class services.
- Enhance onboard entertainment and comfort.
